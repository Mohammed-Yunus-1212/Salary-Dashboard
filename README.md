# Salary Dashboard

## Overview
This project contains a Salary Dashboard created in Excel. It provides insights into salary data through interactive visualizations and analysis.

## Features
- Interactive charts and graphs
- Salary trends analysis
- Department-wise breakdowns
- Employee compensation insights

## Dataset
The dashboard is built using salary data that includes:
- Employee information
- Salary details
- Department and role data
- Performance metrics

## How to Use
1. Download the `Salary Dashboard.xlsx` file
2. Open it in Microsoft Excel or compatible software
3. Enable macros if prompted (if any)
4. Interact with the dashboard using slicers and filters

## Requirements
- Microsoft Excel 2016 or later
- Or compatible spreadsheet software
